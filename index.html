Česká verze (Czech Version)

Tato stránka vás informuje o zásadách týkajících se shromažďování, používání a zveřejňování osobních údajů při používání aplikace Modrá žába.

1. Shromažďování a používání informací

Aplikace Modrá žába byla navržena s důrazem na maximální soukromí uživatele.

  • Žádné osobní údaje: Aplikace neshromažďuje, neodesílá ani neukládá žádné osobní údaje (např. jméno, e-mail, poloha) na žádné externí servery.
  • Místní ukládání: Veškerá data vložená do aplikace (např. deníkové záznamy, nálady, cíle) jsou ukládána výhradně lokálně ve vašem zařízení pomocí technologie localStorage.
  • Přístup k internetu: Aplikace nevyžaduje přístup k internetu pro svou základní funkčnost. Internetové připojení může být použito pouze při otevření externího odkazu, který uživatel sám vložil.

2. Sdílení dat s třetími stranami

Protože aplikace neshromažďuje žádná data, žádná data nejsou sdílena, prodávána ani předávána třetím stranám.

3. Zabezpečení

Aplikace obsahuje volitelnou funkci zabezpečení pomocí PIN kódu. Tento PIN kód je uložen pouze lokálně ve vašem zařízení.

4. Práva uživatele

Uživatel má plnou kontrolu nad svými daty. Data lze kdykoliv smazat pomocí funkce „Resetovat data“ nebo odinstalováním aplikace, čímž dojde k odstranění místního úložiště.

5. Změny těchto zásad

Tyto zásady ochrany soukromí mohou být čas od času aktualizovány. Změny jsou účinné okamžikem jejich zveřejnění na této stránce.

6. Kontakt

V případě dotazů nás můžete kontaktovat na e-mailové adrese: martin@hrosek.eu

English Version

This page describes the policies regarding the collection, use, and disclosure of personal information when using the Modrá žába mobile application.

1. Information Collection and Use

The Modrá žába application is designed with a strong focus on user privacy.

  • No Personal Data Collection: The application does not collect, transmit, or store any personal data (such as name, email address, or location) on external servers.
  • Local Storage Only: All data entered into the application (such as diary entries, moods, or goals) is stored exclusively on the user's device using localStorage technology.
  • Internet Access: The application does not require internet access for its core functionality. Internet connectivity may only be used when the user chooses to open an external link.

2. Data Sharing with Third Parties

Since the application does not collect any data, no data is shared, sold, or transferred to third parties.

3. Security

The application includes an optional PIN protection feature. This PIN is stored locally on the user's device.

4. User Rights

Users have full control over their data. Data can be deleted at any time using the “Reset Data” function or by uninstalling the application, which removes local storage data.

5. Changes to This Privacy Policy

This Privacy Policy may be updated from time to time. Any changes become effective immediately upon being posted on this page.

6. Contact

If you have any questions regarding this Privacy Policy, you may contact us at: martin@hrosek.eu
